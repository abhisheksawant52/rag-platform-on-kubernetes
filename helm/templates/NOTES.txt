Thank you for installing {{ .Chart.Name }} (version {{ .Chart.AppVersion }}).

Release:   {{ .Release.Name }}
Namespace: {{ .Release.Namespace }}

1. Check that the pods are ready:

   kubectl get pods -n {{ .Release.Namespace }} \
     -l app.kubernetes.io/instance={{ .Release.Name }}

{{- if .Values.ingress.enabled }}

2. The service is exposed via Ingress at:
{{- range .Values.ingress.hosts }}
   https://{{ .host }}
{{- end }}
{{- else }}

2. Port-forward to reach the service locally:

   kubectl port-forward -n {{ .Release.Namespace }} \
     svc/{{ include "rag-platform.fullname" . }} 8000:{{ .Values.service.port }}
{{- end }}

3. Probe the API:

   curl http://localhost:8000/health
   curl http://localhost:8000/ready

Metrics are scraped from /metrics on container port 8000.
